## 我遇到了什么问题？

实验过程中主要遇到了以下问题：

1. 直接运行Jupyter Notebook中的二维二分类代码时出现：

   ```text
   NameError: name 'torch' is not defined
   ```

   原因是没有先运行导入依赖的代码单元。

2. 出现：

   ```text
   ModuleNotFoundError: No module named 'pandas'
   ```

   原因是当前 Python 环境缺少相关依赖库。

3. VS Code 使用的 Jupyter Kernel 与安装依赖的 Python 环境不一致，导致安装完成后仍然提示缺少模块。

4. 对 Tensor、Batch、Logit、Loss、数据 Shape 等概念理解不够清晰。

---
